# Proyecto: Universidades (gestor de carreras)

Estructura: `Argentina / Provincia / Facultad` -> CSV(s) con carreras.

Cómo ejecutar:

1. Descomprimir y situarse en la carpeta `universidades_project`.
2. Ejecutar `python main.py` en la terminal o correr codigo como "Python File".
3. Usar el menú para listar, añadir, editar y eliminar carreras.

Archivos incluidos:

- main.py : programa principal interactivo (CLI)
- Argentina/... : estructura con CSVs

Notas:

- Validaciones básicas: duracion_anios y cupos_anuales deben ser enteros > 0. Modalidades válidas: ['Mixta', 'Presencial', 'Virtual'].
